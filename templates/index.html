About the Project

Turning your skills into a clear career path

The Smart Career Recommendation System is a final-year Computer Science project that uses Machine Learning to help students discover careers that genuinely fit them. Instead of guessing, students get a data-driven recommendation.

The system studies academic performance, technical and soft skills, interests, and personality traits, then applies a trained Random Forest Classifier to predict the most suitable career from ten in-demand professions.

✓ Built with Python, Flask, and Scikit-learn
✓ Trained on academic scores, skills & personality data
✓ Instant, explainable career predictions with match %
